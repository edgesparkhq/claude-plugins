# EdgeSpark Claude Code Plugins

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](./LICENSE)

Official Claude Code plugin marketplace by EdgeSpark.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [edgespark](./plugins/edgespark) | Build full-stack web apps on EdgeSpark with database, auth, storage, and deployment workflows |

## Installation

### 1. Add the marketplace (one-time)

```bash
claude plugin marketplace add edgesparkhq/claude-plugins
```

### 2. Install a plugin

```bash
claude plugin install edgespark@edgespark-claude-plugins
```
